先求出一个初始基本可行解，判断其是否最优，若不是最优，再换一个基本可行解并判断，直到得出最优解 / 无最优解

  1. 化线性规划模型为标准型，求初始基本可行解，建立初始单纯形表
  2. 求检验数并判断，若得到最优解，结束计算，否则进行下一步
  3. 基变量，构建新的单纯形表进行迭代
  4. 重复步骤二、三，直到得出最优解 / 无最优解

# 判断检验数（针对目标函数求 maxmaxmax，且无人工变量的情况）

  • 所有检验数都满足 σj≤0\sigma_j\le0σj​≤0，得到最优解，其中
    • 若所有非基变量的检验数均小于零，则为唯一最优解
    • 若存在非基变量的检验数为零，则为多重解
  • 若存在检验数 σk>0\sigma_k>0σk​>0，且其对应的变量 xkx_kxk​ 的系数列向量 Pk≤0P_k\le0Pk​≤0，则为无界解

  • 两阶段单纯型法

    • 第一阶段
      • 决策变量：原问题未知变量 + 松弛变量 + 人工变量
      • 目标函数
        • 单倍人工变量和 xi+...+xjx_i+...+x_jxi​+...+xj​ 求最小值 →\to→ 在找入基变量的时候，找的是最小值入、检验数 ≥0\ge 0≥0 、θi\theta_iθi​ 找最小值
        • 单倍人工变量和 −1∗(xi+...+xj)-1*(x_i+...+x_j)−1∗(xi​+...+xj​) 求最大值 →\to→ 在找入基变量的时候，找的是最大值入、检验数 ≤0\le 0≤0 、θi\theta_iθi​ 找最大值
      • 约束条件：原问题加入人工变量后的约束条件
      • 若得到最优目标函数为零，说明原问题有基可行解，可以进行第二阶段计算；反正，说明原问题无可行解，停止计算
    • 第二阶段
      • 决策变量：原问题未知变量 + 松弛变量
      • 目标函数：原问题目标函数
      • 约束条件：第一阶段计算得到的最终表，去除人工变量列

# 几种特殊解

  • 多重解：存在非基变量的检验数为零
    • 求得一个最优解 x1∗x_1^*x1∗​ 后，以检验数为零的非基变量为进基变量，再进行一次单纯形法迭代运算得到 x2∗x_2^*x2∗​
    • 由 x1∗,x2∗x_1^*,~x_2^*x1∗​, x2∗​ 的线性组合 x∗=ax1∗+(1−a)x2∗,0≤a≤1x^*=ax_1^*+(1-a)x_2^*,~0 \le a \le 1x∗=ax1∗​+(1−a)x2∗​, 0≤a≤1
  • 无界解
    • 极大值问题：存在检验数 σk>0\sigma_k > 0σk​>0，其对应变量 xkx_kxk​ 的系数列向量 aika_{ik}aik​ 均 ≤0\le 0≤0
    • 极小值问题：存在检验数 σ<0\sigma<0σ<0，其对应变量 xkx_kxk​ 的系数列向量 aika_{ik}aik​ 均 ≤0\le 0≤0
  • 退化解：模型中存在多余的约束条件
    • 用最小 θi\theta_iθi​ 比值确定出基变量时候，存在两个及以上的最小比值
  • 不可行解：模型中有相互矛盾的约束条件
    • 基变量中含有非零的人工变量
