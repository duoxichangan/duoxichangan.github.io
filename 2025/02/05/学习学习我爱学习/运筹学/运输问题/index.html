# 表上作业法

  1. 求初始基本可行解（西北角法、最小元素法、伏格尔法）
  2. 求检验数并判断（闭回路法、位势法）
  3. 调整变量（闭回路法）
  4. 重复步骤二、三，直到求得最优解
