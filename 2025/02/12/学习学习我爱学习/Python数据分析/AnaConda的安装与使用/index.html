# AnaConda 是什么

AnacondaAnacondaAnaconda，是专注于数据分析的 PythonPythonPython 发行版本，包含了 condacondaconda、PythonPythonPython 等 190190190 多个科学包及其依赖项。
AnacondaAnacondaAnaconda 就是可以便捷获取包且对包能够进行管理，包括了 pythonpythonpython 和很多常见的软件库和一个包管理器 condacondaconda。常见的科学计算类的库都包含在里面了，使得安装比常规 pythonpythonpython 安装要容易，同时对环境可以统一管理的发行版本。

  • 解释器：运行代码
  • 包集合： 包集合中包含了自带的包和第三方包，第三方包我们一般通过 pippippip 或者 easy_installeasy\_installeasy_install 来下载

# 包管理

如果在本地只有一个 pythonpythonpython 环境，那所有程序用到的各种包都只能放到同一个环境中，会导致环境混乱。另外，当将写好的程序放到另一电脑上运行时，又会遇到缺少相关包，需要手动一个个下载。要是能每个程序开发都选用不同的环境，而开发好之后又能将该程序需要的环境（第三方包）都独立打包出来就好了

  • AnaCondaAnaCondaAnaConda：包含 180+180+180+ 的科学包及其依赖项的发行版本。可以创建一个独立的 pythonpythonpython 环境，其包含的科学包包括：condacondaconda，numpynumpynumpy，scipyscipyscipy，ipythonipythonipython，notebooknotebooknotebook 等
  • VirtualenvVirtualenvVirtualenv：用于创建一个独立的 pythonpythonpython 环境的工具，新环境中需要手动安装需要的第三方包

# AnaConda 的好处

AnaCondaAnaCondaAnaConda 对于初学者而言及其友好，相比单独安装 pythonpythonpython 主程序，选择 AnaCondaAnaCondaAnaConda 可以帮助省去很多麻烦，AnaCondaAnaCondaAnaConda 里添加了许多常用的功能包，如果单独安装 pythonpythonpython，这些功能包则需要一条一条自行安装，在 AnaCondaAnaCondaAnaConda 中则不需要考虑这些，同时 AnacondaAnacondaAnaconda 还附带捆绑了两个非常好用的交互式代码编辑器（ SpyderSpyderSpyder、JupyternotebookJupyter~notebookJupyter notebook ）。

在 pythonpythonpython 中，我们写程序的时候经常离不开第三方库，我们可以称之为包。包可以理解成一个工具，我们要通过这个工具去实现我们所需要的功能。怎样使用这个工具呢？毫无疑问，我们都是 import…, import…

但是，importimportimport 初始只能导入 pythonpythonpython 自带的库，第三方库需要我们自己通过 cmdcmdcmd 去 pipinstallpip~installpip install。因为很多库都是 pythonpythonpython 不自带的，我们都是要通过 pippippip 额外安装。
这个时候 AnaCondaAnaCondaAnaConda 的作用就出来了，AnaCondaAnaCondaAnaConda 自带了很多 pythonpythonpython 库，不用我们自己去额外安装，极少数我们需要安装的时候，我们通过 conda install xxx 就好了，及其方便！

# 什么是 Conda

condacondaconda 是一个包管理器，环境管理器。
上文说到，AnaCondaAnaCondaAnaConda 可以创建一个虚拟的 pythonpythonpython 环境（与主环境分离），而 AnaCondaAnaCondaAnaConda 正是通过 condacondaconda 方法创建虚拟环境。
