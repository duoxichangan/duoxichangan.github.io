# 导入库

  • plot() ：用于绘制线图和散点图
  • scatter() ：用于绘制散点图
  • bar() ：用于绘制垂直条形图和水平条形图
  • hist() ：用于绘制直方图
  • pie() ：用于绘制饼图
  • imshow() ：用于绘制图像
  • subplots() ：用于创建子图

例如，绘制一条线

# plot

plot() 函数是绘制二维图形的最基本函数，可以绘制点和线，语法如下：

  • x,yx,yx,y：点或线的结点
  • fmtfmtfmt：定义基本格式（颜色、标记和线条样式等）
  • ∗∗kwargs**kwargs∗∗kwargs：用在二维平面上，设置指定属性（标签、线的宽度等）


如果只想绘制点，那就选择标记字符 plt.plot(x, y, 'o')
