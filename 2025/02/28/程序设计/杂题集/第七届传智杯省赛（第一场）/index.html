问：给定 qqq 个询问，每次询问一个 ddd，问你第 ddd 天摘完后，还剩多少个

# 小苯的数组最值 (A 组、B 组、C 组)

# 题意

有一个长度为 nnn 的数组
对这个数组施展 mmm 次魔法，每次可以对于给定的 (l,r,d)(l, r, d)(l,r,d)，让其中的 al→ara_l \to a_ral​→ar​ 都加上 ddd
但是，不会全部施法成功，而是会随机且等概率地失效一个魔法
问，在这种情况下，数组 aaa 的最大值的期望是多少
（对于每个测试数据，输出一行一个整数表示数组的最大值对 998244353998244353998244353 取模的值

# 题解

求一个区间的最大值，引入前缀 + 后缀最大值数组，修改值用差分来记录，注意，这题用到了快速幂和逆元

期望的计算公式是： 期望 = 所有可能情况总和 / 情况数
在此题中是 期望 = ans / q

在除运算中，不能直接模
(a / b) % c != (a % c) / (b % c)

这时就要用到逆元
a / b ≡ a * b^{-1} (mod p)

b^{-1} 代表 bbb 的逆元，计算逆元用到了费马小定理
如果 ppp 是质数，且 ppp 和 qqq 互质，则 q^{p-2} ≡ q^{-1} (mod p)
因此，qp−2%pq^{p-2} \% pqp−2%p 返回 qqq 的逆元

前缀和 二分 模拟 小学数学 预处理 逆元 快速幂
