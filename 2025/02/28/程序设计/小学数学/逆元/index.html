# 同余

两个整数 a,ba, ba,b，如果(a−b)%m=0(a - b) \% m = 0(a−b)%m=0，则称 a、ba、ba、b 对于模 mmm 同余，记作 a≡b(modm)a ≡ b(mod~m)a≡b(mod m)，读作：aaa 同余于 bbb 模 mmm
a≡b(moda ≡ b(moda≡b(mod m)m)m) →\to→ (a−b)(a - b)(a−b) %\%% m==0m == 0m==0

# 何谓 "逆元"？

在取模的条件下，除以一个数等于乘以这个数的乘法逆元

x/2%107≡x∗54%107x / 2 \% 107 ≡ x* 54 \% 107x/2%107≡x∗54%107

那么 545454 就是 222 在模 107107107 条件下的乘法逆元
一个数 aaa 存在乘法逆元的充要条件是 aaa 与模数 mmm 互质。
当模数 mmm 为质数时，am−2a^{m−2}am−2 即为 aaa 的乘法逆元
