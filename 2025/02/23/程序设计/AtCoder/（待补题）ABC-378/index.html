# 题意

# 题解

证明

方程等价于： d + x = k * q[t] + r[t]
x = k * q[t] + r[t] - d
因为 x≥0x \ge 0x≥0，所以可以得到

k⋅q[t]+r[t]−d≥0k · q[t] + r[t] - d \ge 0k⋅q[t]+r[t]−d≥0

等价于

k≥d−r[t]q[t]m,minn(k)=⌈d−r[t]q[t]⌉k \ge \frac{d - r[t]}{q[t]}m,minn(k) = \lceil{\frac{d - r[t]}{q[t]}}\rceilk≥q[t]d−r[t]​m,minn(k)=⌈q[t]d−r[t]​⌉

可以写成

k=d−r[t]+q[t]−1q[t]k = \frac{d-r[t]+q[t]-1}{q[t]}k=q[t]d−r[t]+q[t]−1​

总结有 ⌈ab⌉\lceil{\frac{a}{b}}\rceil⌈ba​⌉ = a+b−1b\frac{a+b-1}bba+b−1​
