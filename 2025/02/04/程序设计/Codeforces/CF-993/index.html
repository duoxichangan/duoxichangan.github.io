# C

# 题意

一个教室有 mmm 列，222 行座位
三种猴子

  1. AAA 猴子只坐第一排
  2. BBB 猴子只坐第二排
  3. CCC 猴子随便坐

问：最多能给多少只猴子上课

# 题解

只需要先行排 AAA 和 BBB 猴子，剩下的若还有空位，插 CCC 猴子即可
