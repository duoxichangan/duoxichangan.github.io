# 内置 sortsortsort 排序

内置 sortsortsort 采取的是混合排序算法，均摊时间复杂度 O(nlog⁡n)O(n \log{n})O(nlogn)

# 快速排序

  • 最好时间复杂度 O(nlog⁡n)O(n \log{n})O(nlogn)
