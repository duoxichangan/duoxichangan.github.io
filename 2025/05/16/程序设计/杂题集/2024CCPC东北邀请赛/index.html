# L - Recharge

# 题目大意

你现在有一个电池，这块电池有 kkk 格电，现在，有两种充电站

  • xxx 个第一种充电站，可以加 111 格电
  • yyy 个第二种充电站，可以加 222 格电

若你现在有 k−1k-1k−1 格电，那么使用第二种充电站的话，则会充满这块电池但是就浪费了一格电
问：最多能充满多少块这样的电池？

# 数据范围

  • ttt 组数据，1≤t≤2×1051 \le t \le 2 \times 10^51≤t≤2×105
  • 1≤k≤1091 \le k \le 10^91≤k≤109
  • 0≤x,y≤1090 \le x,y \le 10^90≤x,y≤109

# 题解

尽量先用加 222 格电的充电站先充，针对 kkk 的奇偶性进行分类讨论

  • kkk 是偶数
    • 那么需要消耗 t = k / 2 个 222 即可拼成一个 kkk
    • 所以，只用 222 的话，可以凑出 a = y / t 个，还剩下 y = y - a * t 个 222
    • 那么剩下的就可以直接散着拼了，也就是有 (x + 2 * y) / k 个
    • 最终答案为 a + (x + 2 * y) / k ，此处 yyy 为减之后的 yyy
  • kkk 是奇数
    • 那么需要消耗 t = (k - 1) / 2 个 222 和一个 111 即可拼成一个 kkk
    • 所以可以先拼出 a = min(y / t, x) 个，还剩下 y = y - a * t 个 222 和 x = x - t 个 111
    • 剩下的散着拼，但是这里还要考虑到浪费电的那种充电情况
      • 如果 111 的还有剩余，那么和前面的情况，也就是有 (x + 2 * y) / k 个
        • 那么最后答案就是 a + (x + 2 * y) / k ，此处的 xxx 和 yyy 均为减之后的 xxx 和 yyy
      • 如果 111 的没有剩余，那么就全部用 222 拼，只能考虑浪费，也就是有 y / ((k + 1) / 2) 个
        • 那么最后答案就是 a + y / ((k + 1) / 2) ，此处的 xxx 和 yyy 均为减之后的 xxx 和 yyy

这题还有一个坑是，kkk 的范围是 1≤k≤1091 \le k \le 10^91≤k≤109，那么当 k=1k = 1k=1 时，我们前面讨论中的除 k−1k - 1k−1 就会 RERERE
再添加一个特判，当 k=1k=1k=1 时，答案是 x+yx+yx+y
