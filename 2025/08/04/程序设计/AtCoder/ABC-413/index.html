# 题目大意

给你一个 A=(a1,...,an)A=(a_1,...,a_n)A=(a1​,...,an​)，问你是否可以重排列成 B=(b1,...,bn)B=(b_1,...,b_n)B=(b1​,...,bn​) 使得 BBB 是一个等比数列

# 数据范围

  • 1≤T≤1051 \le T \le 10^51≤T≤105
  • 2≤N≤2×1052 \le N \le 2 \times 10^52≤N≤2×105
  • −109≤ai≤109-10^9 \le a_i \le 10^9−109≤ai​≤109

# 题解

AAA 可能有：

  • 纯正数
  • 纯负数
  • 正数负数交替

需要注意的是，正数负数交替的情况

  • 必须满足， max(正数个数, 负数个数) - min(正数个数, 负数个数) <= 1
  • 注意，等比数列的公比 DDD 的绝对值，可能 ≥1\ge 1≥1 也可能 <1< 1<1
    • 如果 |D| >= 1 的话，那么序列绝对值应该是升序的
    • 如果 |D| < 1 的话，那么序列绝对值应该是降序的
