# C

# 题目大意

一排有 NNN 个格子（初始时全部是白色）
有 QQQ 个查询，每个查询给出一个 aia_iai​，你需要

  • 翻转从左数，第 aia_iai​ 个格子的颜色，之后，统计当前所有连续的黑色格子区间的数量

# 数据范围

  • 1≤N,Q≤5×1051 \le N, Q \le 5 \times 10^51≤N,Q≤5×105

# 题解

注意，题目是翻转

  • 如果是涂黑
    • 如果左边右边均没有，那答案就 +1+1+1
    • 如果只有一边有，答案不变
    • 如果两边都有，答案 −1-1−1
  • 如果是涂白
    • 如果左边右边均没有，答案 −1-1−1
    • 如果左边右边都有 答案 +1+1+1
    • 如果只有一边有，答案不变

# D

# 题目大意

你有一台服务器和 nnn 台电脑，他们都各自持有一个字符串，初始时，均为空
接下来有 QQQ 个操作，每个操作是下面三种之一

  1. 1 p ：将第 ppp 台电脑的字符串用服务器字符串替代
  2. 2 p s ：在第 ppp 台电脑的字符串末尾追加字符串 sss
  3. 3 p ：将服务器字符串用第 ppp 台电脑的字符串替换

请你输出所有操作后的，服务器字符串

# 数据范围

  • 1≤n,Q≤2×1051 \le n,Q \le 2 \times 10^51≤n,Q≤2×105
