# C

# 题目大意

给定 aaa 和 nnn，请问 1→n1 \to n1→n 中，有多少个数，在十进制和 aaa 进制下同时都是回文数

# 数据范围

  • 2≤a≤92 \le a \le 92≤a≤9
  • 1 \le n \le 10^

# 题解

这题显然不能暴力枚举 1→n1 \to n1→n，可以发现，101210^{12}1012 内的回文数大概有 10610^{6}106 数量级的回文数，不妨考虑预处理好，然后遍历判断其是否是 aaa 进制

如果要预处理 NNN 以内的回文数，可以遍历 1→N1 \to \sqrt{N}1→N​，通过奇数、偶数长度的反转来实现预处理回文数

# D

# 题目大意

现在有 nnn 座房子，分别位于 x1→xnx_1 \to x_nx1​→xn​，你需要在数轴上任意放置 mmm 所基站，基站的信号强度如果是 ttt 的话，那它只能影响到左右距离不超过 t2\frac{t}{2}2t​ 的房子，请问，信号强度和的最小值是多少

# 数据范围

  • 1≤M≤N≤5×1051 \le M \le N \le 5 \times 10^51≤M≤N≤5×105
  • 1 \le x_i \le 10^

# 题解

显然，在最优情况下，基站的两个边界是都有房子的，那么题目要安排 mmm 个基站，也就是说，会把 min⁡i=1i=nai→max⁡i=1nai\min\limits_{i=1}^{i=n} a_i \to \max\limits_{i=1}^{n} a_ii=1mini=n​ai​→i=1maxn​ai​，以 aia_iai​ 为分割，分成了 mmm 个块
我们的需求是使得 ∑t\sum t∑t 最小，也就是我们可以让剩余的 m−1m-1m−1 个空的长度尽可能大，这样减出来的 ∑t\sum t∑t 就一定会更小

# E

# 题目大意

计算满足以下条件的整数三元组 (a,b,c)(a, b,c)(a,b,c) 的个数对 998244353998244353998244353 取模的结果

  • 1≤a,b,c≤N1 \le a, b, c \le N1≤a,b,c≤N
  • a,b,ca,b,ca,b,c 互不相同
  • a%b=ca \% b = ca%b=c

# 数据范围

  • 3 \le N \le 10^

# 题解

# 1. 确定 b 的范围

题目要求 a,b,ca,b,ca,b,c 两两不相同，我们可以讨论一下他们的大小关系

  • a<ba < ba<b 时， a % b = a ，不满足三者不相同
  • a>ba > ba>b 时， a % b = c ，此时 N≥a>bN \ge a > bN≥a>b，只要满足 b>cb > cb>c，则三者必然两两不相同

由 N≥a>b>c≥1N \ge a > b > c \ge 1N≥a>b>c≥1 有，b∈[2,N−1]b \in [2, N - 1]b∈[2,N−1]

# 2. 计算 a 的个数

由 N≥a>b>c≥1,b∈[2,N−1]N \ge a > b > c \ge 1,b \in [2, N - 1]N≥a>b>c≥1,b∈[2,N−1] 有，a∈[b+1,N]a \in [b+1, N]a∈[b+1,N]
因为 a%b=c≥1a \% b = c \ge 1a%b=c≥1，所以要排除 aaa 是 bbb 的倍数
也就是

a=kb,k∈[2,⌊Nb⌋]a = kb,k \in [2, \lfloor{\frac{N}{b}}\rfloor]a=kb,k∈[2,⌊bN​⌋]

所以，aaa 的个数是

(N−(b+1)+1)−(⌊Nb⌋−2+1)=N−b−⌊Nb⌋+1(N-(b+1)+1)-(\lfloor{\frac{N}{b}}\rfloor-2+1)=N-b-\lfloor{\frac{N}{b}}\rfloor+1(N−(b+1)+1)−(⌊bN​⌋−2+1)=N−b−⌊bN​⌋+1

# 3. 求答案

也就是说，答案是求

∑b=2N−1N−b−⌊Nb⌋+1\sum\limits_{b=2}^{N-1} N-b-\lfloor{\frac{N}{b}}\rfloor+1b=2∑N−1​N−b−⌊bN​⌋+1

显然，这样的话复杂度就爆炸了

∑b=2N−1(N−b+1)−∑b=2N−1(⌊Nb⌋)\sum\limits_{b=2}^{N-1}(N-b+1)-\sum\limits_{b=2}^{N-1}(\lfloor{\frac{N}{b}}\rfloor)b=2∑N−1​(N−b+1)−b=2∑N−1​(⌊bN​⌋)

左侧可以用等差数列计算和，右侧可以用整除分块

# 整除分块

⌊Nb⌋\lfloor{\frac{N}{b}}\rfloor⌊bN​⌋ 在 bbb 的某些连续区间内的取值是相同的，我们可以按照 ⌊Nb⌋\lfloor{\frac{N}{b}}\rfloor⌊bN​⌋ 的值来对 bbb 进行分块，贡献是 长度 * 值

对于一个分块的起点 lil_ili​ 而言，其下一个分块的起点应该满足

li+1=min⁡{x∣x>li,⌊N/x⌋<k}l_{i+1}=\min \{x | x > l_i, \lfloor{N/x}\rfloor < k\}li+1​=min{x∣x>li​,⌊N/x⌋<k}

⌊N/x⌋<k\lfloor{N/x}\rfloor < k⌊N/x⌋<k 有 L>⌊N/k⌋L > \lfloor{N/k}\rfloorL>⌊N/k⌋，所以 l[i+1] = min(n / k + 1, N)

遇到 ∑i=1n⌊常数i⌋\sum\limits_{i=1}^{n} \lfloor{\frac{常数}{i}}\rfloori=1∑n​⌊i常数​⌋ 时，可以考虑用整数分块的思想，整数分块求和的时间复杂度是 O(n)O(\sqrt{n})O(n​) 的
