# C1 - Easy Version

# 题目大意

现在买 nnn 个东西，每次你可以选择一个非负整数 xxx，以 3x+1+x×3x−13^{x + 1} + x \times 3^{x - 1}3x+1+x×3x−1 的价钱，购买 3x3^x3x 个物品，请问，如果要使得交易次数最小的话，你要花多少钱？

# 数据范围

  • 1≤n≤1091 \le n \le 10^91≤n≤109

# 题解

显然，按照 nnn 的三进制来购买物品是能满足交易次数最小的

# C2 - Hard Version

# 题目大意

在 C1C1C1 题的基础上，增加一个正整数 kkk，题目要求变为，在不超过 kkk 次交易的情况下，最少花费多少能购买到 nnn 个物品

# 题解

显然，我们在 C1C1C1 里知道了，最少的交易次数是按 nnn 的三进制来买，所以如果我们的 kkk 小于三进制下的各位之和，肯定是不能购买的，输出 -1
题目中说了，3x3^x3x 个物品花费 3x+1+x×3x−13^{x + 1} + x \times 3^{x - 1}3x+1+x×3x−1，也就是说，针对 xxx 的话，单价是 3+x33 + \frac{x}{3}3+3x​，随着 xxx 越大，货物单价也越大
