# B

# 题目大意

有两个网格 S,TS, TS,T，每个网格都是 NNN 行 NNN 列
每个单元格都被涂成白 / 黑色，单元格是 . 就是白色，是 # 就是黑色
你可以进行下面两种操作任意次

  • 选中 SSS 中的一个格子，反转颜色
  • 将 SSS 顺时针旋转 90°90°90°

请问至少要多少次操作，才能使 SSS 和 TTT 变成一样的？

# 数据范围

  • 1≤n≤1001 \le n \le 1001≤n≤100

# 题解

考虑四种旋转后，与 TTT 图的差异即可，需要注意的是，这四次旋转需要的 222 操作次数是不一样的

# C

# 题目大意

给你一个 NNN 个点 MMM 条边的无向图，请你判断这个图是否是循环图

循环图的定义
对于一个图，若存在 (1,2,...,N)(1,2,...,N)(1,2,...,N) 的一个排列 (v1,v2,...,vN)(v_1,v_2,...,v_N)(v1​,v2​,...,vN​) 满足

  1. 对于每个i=1,2,...,N−1i = 1,2,...,N-1i=1,2,...,N−1，viv_ivi​ 和 vi+1v_{i+1}vi+1​ 之间有一条边
  2. v1v_1v1​ 和 vNv_NvN​ 直接有一条边
  3. 不存在其他边

那么就称这个图为循环图

# 数据范围

  • 3≤N≤2×1053 \le N \le 2 \times 10^53≤N≤2×105
  • 0≤M≤2×1050 \le M \le 2 \times 10^50≤M≤2×105

# 题解

3. 不存在其他边 这个条件，要求此时 n == m
如果是一个环图，那么每个点一定只有两个方向能走，也就是说， g[i].size()==0 ；并且任意从一点开始深搜，如果能搜到 nnn 个点（在 n == m 的前提条件下），那么一定是个环图

# E

# 题目大意

你有 NNN 个茶碗，它们排成一排，编号为 0,1,...,N−10,1,...,N-10,1,...,N−1
对于 111 号碗和其右边的碗，第 iii 个碗上写着 CiC_iCi​，里面装了 AiA_iAi​ 个豆子，000 号碗没写数字，没装豆子
你将执行以下操作若干次

  • 选择一个写了数字的、有豆子的碗，拿走若干个豆子
  • 把拿出的豆子任意放入 i−Ci,i−Ci+1,...,i−1i-C_i,i-C_i+1,...,i-1i−Ci​,i−Ci​+1,...,i−1 号碗里

请问，将所有豆子放入 000 号碗，最少需要操作多少次？

  • 1≤N≤20001 \le N \le 20001≤N≤2000
