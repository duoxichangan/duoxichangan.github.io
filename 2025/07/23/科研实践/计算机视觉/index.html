# 中值滤波

祛痘是美颜时常用的操作，痘 →\to→ 椒盐噪声，高斯滤波不能有效地处理，可以用中值滤波
中值滤波器是一种统计排序滤波器，图像像素等于周围像素排序后的中值，可以有效抑制椒盐噪声
