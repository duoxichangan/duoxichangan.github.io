# D

# 题目大意

给你一个长度为 nnn 的，只有 0,10,10,1 构成的字符串 sss，你可以执行以下操作若干次

  • 选择任意一个字符，Si←1−SiS_i \leftarrow 1-S_iSi​←1−Si​

请问，至少经过多少次操作，字符串的 111 连续或不存在 111

# 数据范围

  • 1≤T≤2×1041 \le T \le 2 \times 10^41≤T≤2×104
  • 1≤∣S∣≤2×1051 \le |S| \le 2 \times 10^51≤∣S∣≤2×105

# 题解

如果最终，是 [l,r][l,r][l,r] 区间连续的 111 的话，其操作次数为

(r−l+1−(p[r]−p[l−1]))⏟区间里要翻的 0 数+(p[l−1]+(p[n]−p[r]))⏟区间外要翻的 1 数,\underbrace{\bigl(\,r-l+1 - \bigl(p[r]-p[l-1]\bigr)\bigr)}_{\text{区间里要翻的 0 数}} +\underbrace{\bigl(p[l-1] + (p[n]-p[r])\bigr)}_{\text{区间外要翻的 1 数}},区间里要翻的 0 数(r−l+1−(p[r]−p[l−1]))​​+区间外要翻的 1 数(p[l−1]+(p[n]−p[r]))​​,

化简有

p[n]+(r−2p[r])+(2p[l−1]−(l−1))p[n]+(r-2p[r])+(2p[l-1]-(l-1))p[n]+(r−2p[r])+(2p[l−1]−(l−1))

这样看来，还是有两个变量，枚举需要 n2n^2n2 时间，怎么办呢？
我们不放想想，我们求的是答案的最小值，那我们可以只遍历 rrr，对于一个确定的 rrr，其是不是只要找到，min⁡i=1r−1(2p[l−1]−(l−1))\min\limits_{i=1}^{r-1} (2p[l-1]-(l-1))i=1minr−1​(2p[l−1]−(l−1)) 呢，而这个我们就可以用前缀最小值来实现

这题提示我们，如果出现了区间最小 / 最大操作次数的题目，我们不妨从暴力枚举 [l,r][l,r][l,r] 区间的地方开始着手
利用前缀和或其他知识，写出最终区间是 [l,r][l, r][l,r] 的操作数的表达式，可以猜想到，最终表达式一定是一个关于 [l,r][l, r][l,r] 的方程
此时不妨尝试枚举 rrr，进而观察 lll 相关的式子，比如此题中，要求的是求最小值，而 lll 相关的式子可以用一个前缀最小值来简化掉一层循环
