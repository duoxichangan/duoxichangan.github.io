# M

# 题目大意

现在给你 nnn 个点 mmm 条边，你可以选择创建或销毁一条通路，请问，最少需要经过多少次操作后，图可以变成一个无向树

# 数据范围

  • 1≤n≤1061 \le n \le 10^61≤n≤106
  • 0≤m≤1060 \le m \le 10^60≤m≤106

# 题解

这题还挺容易漏考虑情况的，首先，很容易想到的情况是

  • 重遍
  • 自环

这可以很容易用并查集解决，只要新来的边的两个点，祖宗节点相同，则不允连边

那么还有以下两种情况

  • 孤立点
  • 多个孤立集合

为什么是两种呢，因为孤立点也可以看作一个孤立集合，显然，在多个孤立集合（每个集合都是一个无向树）之间，要使得整个图变成一个无向树，只需要再多连 孤立集合个数 - 1 条边即可，赛时一把过，赛后 WAWAWA 三发

# H

# 题目大意

初始时，有无穷个孤立的结点，第 iii 个结点的编号为 iii
树论函数的定义为：f(n)=n×(n+1)f(n)=n \times (n+1)f(n)=n×(n+1)，若存在 f(n)=f(a)×f(b)f(n)=f(a) \times f(b)f(n)=f(a)×f(b)，则由结点 nnn 出发，分别向 a,ba,ba,b 建立一条无向边

f(x)×f(x+1)=x(x+1)(x+1)(x+2)f(x) \times f(x+1)=x(x+1)(x+1)(x+2)f(x)×f(x+1)=x(x+1)(x+1)(x+2)

所以，f(x)×f(x+1)=[x(x+2)]×[x(x+2)+1]=f[x(x+2)]f(x) \times f(x+1) = [x(x+2)] \times [x(x+2)+1] = f[x(x+2)]f(x)×f(x+1)=[x(x+2)]×[x(x+2)+1]=f[x(x+2)]，得证

# F

# 题目大意

给定 n×mn \times mn×m 的迷宫，每个格子为 .（空地） 或 #（障碍） ，你从 (1,1)(1, 1)(1,1) 出发，目标是到达右下角 (n,m)(n,m)(n,m)，每次可以上、下、左、右移动一格，你可以至多服用一次药剂，在服用后的连续 kkk 步中，你可以把障碍物视为空地
请你计算，从起点移动到终点，kkk 的最小值是多少

# 数据范围

  • 1≤T≤2.5×1051 \le T \le 2.5 \times 10^51≤T≤2.5×105
  • 2≤n,m≤10002 \le n,m \le 10002≤n,m≤1000
  • 保证 ∑(nm)≤106\sum{(nm)} \le 10^6∑(nm)≤106

# 题解

题目要求，服用药剂后，是连续 kkk 步，所以我们最优的走法，肯定是走到左上角 . 的边缘，然后经过若干个 . / # ，到达右下角 . 的边缘，所以我们可以先做两遍 BFSBFSBFS，把左下角和右下角连续的 . 保留为 . ，其他都视为 #
然后从 (1,1)(1, 1)(1,1) 点开始做 0−1BFS0-1BFS0−1BFS

  • 遇到 . push_frontpush\_frontpush_front，需要的 kkk 为上一步的 kkk
  • 遇到 # push_backpush\_backpush_back，需要的 kkk 为上一步的 k+1k + 1k+1

这题就是我战犯的题目， memset 真的太慢了，从现在开始，只要有频繁 memset 的都换成重新 vectorvectorvector 开，这里对了就铜牌了
