# 合法密码

# 题目大意

密码的要求是这样的

  • 长度大于等于 888 个字符，小于等于 161616 个字符
  • 必须包含至少 111 个数字字符和至少 111 个符号字符

字符串为： kfdhtshmrw4nxg#f44ehlbn33ccto#mwfn2waebry#3qd1ubwyhcyuavuajb#vyecsycuzsmwp31ipzah#catatja3kaqbcss2th
请问有多少个子串可以当作合法密码？

# 立定跳远

# 题目大意

小明从数轴的原点开始向正方向立定跳远，设置了 nnn 个检查点 a1,a2,...,an(ai≥ai−1≥0)a_1,a_2,...,a_n~(a_{i} \ge a_{i-1} \ge 0)a1​,a2​,...,an​ (ai​≥ai−1​≥0)
小明必须先后跳跃到每个检查点上且只能跳跃到检查点上
为了使跳远更加轻松，小明可以自己添加 mmm 个检查点

在运动会前，小明单次跳远最远距离是 LLL
小明可以爆发一次，使得跳跃距离达到 2L2L2L
请问，LLL 的最小值是多少，小明可以完成这个项目

# 数据范围

  • 2≤n≤1052 \le n \le 10^52≤n≤105
  • m≤108m \le 10^8m≤108
  • 0≤ai≤1080 \le a_i \le 10^80≤ai​≤108
