# 视图

视图是虚表，是从若干个基本表（或视图）中导出的表，数据库系统表中仅存放了创建视图的语句，而不存放视图对应的数据

# 视图的作用

  • 简化操作
  • 使用户能从多种角度看待同一数据库模式
  • 对重构数据库模式提供了一定程度的逻辑独立性
  • 能够对数据库的机密数据进行保护
  • 更清晰表达查询
