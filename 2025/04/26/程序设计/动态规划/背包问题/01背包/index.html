# 题目大意

有 nnn 件物品和一个容量是 vvv 的背包，每件物品只能使用一次
第 iii 件物品的体积是 viv_ivi​，价值是 wiw_iwi​
问：你放进背包的最大价值是多少！

# 数据范围

  • 0<N,V≤10000 < N, V \le 10000<N,V≤1000
  • 0<vi,wi≤10000 < v_i, w_i \le 10000<vi​,wi​≤1000

# 思路

# 变量假设

dp[i][j] 表示前 iii 个物品中，已经用了 jjj 的背包体积时的最大价值

# 转移方程

dp[i][j] = max(第 i 个物品放入背包的总价值，第 i 个物品不放入背包的总价值)
放入背包的前提是放得下

# 解题代码

先通过二维的来解释思路

  • 第 iii 个物品要的话，那么新价值就是 dp[i - 1][j - v[i]] + w[i]
  • 第 iii 个物品不要的话，那么新 价值就是 dp[i - 1][j]

    观察有，在求 maxmaxmax 的时候，只用到了上一层的 dp[i - 1][j - v[i]] ，所以可以优化成一维的
