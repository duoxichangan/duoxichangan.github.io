# D

# 题目大意

高桥居住的小镇被划分成一个由 HHH 行和 WWW 列组成的网格。每个单元格要么是一条路，要么是一堵墙
我们把从上面 (1≤i≤H)(1\leq i \leq H)(1≤i≤H) 起第 iii 行，从左边 (1≤j≤W)(1\leq j \leq W)(1≤j≤W) 起第 jjj 列的单元格称为单元格 (i,j)(i,j)(i,j)
每个单元格的信息由长度为 WWW 的 HHH 字符串 S1,S2,...,SHS_1,S_2,...,S_HS1​,S2​,...,SH​ 提供。具体来说：

  • 如果 SiS_iSi​ 的 j−thj-thj−th 字符是 (1≤i≤H,1≤j≤W)(1\leq i \leq H,1\leq j\leq W)(1≤i≤H,1≤j≤W) 是 . ，那么 (i,j)(i,j)(i,j) 单元格就是一条路
  • 如果是 # ，那么 (i,j)(i,j)(i,j) 单元格就是一堵墙

他可以按任意顺序重复执行以下两种操作：

  • 移动到相邻的单元格（向上、向下、向左或向右），该单元格位于小镇内且是一条道路。
  • 从四个方向（上、下、左或右）中选择一个，然后朝该方向踢一脚
    • 当他踢出一脚时，如果该格是一堵墙，则该格就会变成一条路
    • 如果距离最多 222 步的一些单元格在城镇之外，则仍然可以进行前踢，但城镇之外的任何东西都不会改变
      他从 (A,B)(A,B)(A,B) 单元格开始，想要移动到 (C,D)(C,D)(C,D) 单元格的鱼店。
      可以保证他起始的单元格和有鱼店的单元格都是道路。
      求他到达鱼店所需的最少个前踢

# 数据范围

  • 1≤H,W≤10001\leq H,W \leq 10001≤H,W≤1000

# 题解

使用 0−1BFS0-1~BFS0−1 BFS

  • 如果是需要前踢的，就放在队列尾部
  • 如果是可以直接走的，放在队列头部
