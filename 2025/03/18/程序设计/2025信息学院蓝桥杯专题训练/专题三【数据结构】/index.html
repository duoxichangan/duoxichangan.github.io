# Card Pile

有一叠 100100100 纸牌，每张都标有整数 000 。
处理 QQQ 个查询。每个查询都属于以下情况之一：

  • 输入 111 ：将一张标有整数 xxx 的卡片放在堆栈顶部。
  • 类型 222 ：取出牌堆顶端的卡片，并输出写在该卡片上的整数。在此问题的限制条件下，牌堆中总是至少有一张牌。

# 题解

纯粹考栈的操作，会栈 stackstackstack 就会写的题目，无需多言。

# 题解

维护一个单调递减的栈 sss

  • 对于当前元素  a[i]
    • 如果栈  s  不为空且栈顶元素对应的值  a[s.top()]  小于等于  a[i] ，则弹出栈顶元素，从而保证了栈  s  中的元素始终是单调递减的。

# Max × Sum

# 题目大意

给定一个长度为 NNN 的序列： A=(A1,A2,...,AN)A = (A_1, A_2, ..., A_N)A=(A1​,A2​,...,AN​) 和 B=(B1,B2,...,BN)B = (B_1, B_2, ..., B_N)B=(B1​,B2​,...,BN​) 。
设 SSS 是大小为 KKK 的 {1,2,…,N}\lbrace1, 2, \dots, N\rbrace{1,2,…,N} 的子集。求以下表达式的最小可能值：

(max⁡i∈SAi)×(∑i∈SBi)(\max_{i \in S} A_i) \times (\sum_{i \in S} B_i)(i∈Smax​Ai​)×(i∈S∑​Bi​)

给你 TTT 个测试用例，请逐个求解。

# 数据范围

  • 1≤T≤2×1051 \le T \le 2 \times 10^51≤T≤2×105
  • 1≤K≤N≤2×1051 \le K \le N \le 2 \times 10^51≤K≤N≤2×105
  • 1≤Ai,Bi≤1061 \le A_i, B_i \le 10^61≤Ai​,Bi​≤106

# 题解

将 AAA 序列按从小到大的顺序排。
排完序之后，显然，Ai(i∈[1,k−1])A_i~(i \in [1,k-1])Ai​ (i∈[1,k−1]) ，不可能会被选上
那么我们只需要从 AkA_kAk​ 枚举到 AnA_nAn​，这个范围内都是满足题意的、可以被选的 AiA_iAi​.

那么问题来了？∑i∈SBi\sum_{i \in S} B_i∑i∈S​Bi​ 怎么办呢？
我们可以用优先队列维护，先将前 B1→Bk−1B_1 \to B_{k-1}B1​→Bk−1​，pushpushpush 进队列并用一个 sum 累加和。
而对于可以选择的 Ai,(i∈[k,n])A_i,(i \in [k, n])Ai​,(i∈[k,n])，当 AiA_iAi​ 被选中时，BiB_iBi​ 即要加入和，那么这时会有两种情况：

  • heap.size() == k
    • 那么我们就要先弹出一个最大的来，代表不选它。
  • heap.size() == k - 1
