# 全排列问题

# 题目背景

按照字典序输出自然数 111 到 nnn 所有不重复的排列，即 nnn 的全排列，要求所产生的任一数字序列中不允许出现重复的数字。

# 输入格式

一个整数 nnn.

# 输出格式

由 1∼n1 \sim n1∼n 组成的所有不重复的数字序列，每行一个序列。
每个数字保留 555 个场宽。

# 数据范围

1≤n≤91 \leq n \leq 91≤n≤9。

在从顶点 111 到顶点 NNN 的所有简单路径（不经过同一顶点多次的路径）中，求路径上边的标签的最小异或值。

关于位向 XOR 对于非负整数 aaa 和 bbb，a⊕ba \oplus ba⊕b 的定义如下：
在 a⊕ba \oplus ba⊕b 的二进制表示中，当且仅当 aaa 和 bbb 的 2k2^k2k 位上的数字中正好有一位是 111 时， 2k2^k2k 位（ k≥0k \ge 0k≥0 ）上的数字是 111 ；否则，它是 000.
