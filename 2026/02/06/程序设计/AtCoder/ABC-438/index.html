# 题目大意

有 qqq 个询问，每次询问，问 xxx 号水桶，在 yyy 次操作后，有多少水

# 数据范围

  • 2≤n≤2×1052 \le n \le 2 \times 10^52≤n≤2×105
  • 1≤q≤2×1051 \le q \le 2 \times 10^51≤q≤2×105

# 题解

第 iii 个桶，第一次操作后，会在 aia_iai​ 手上，第二次操作后会在 aaia_{a_i}aai​​ 手上 …\ldots…
倍增

  • fi,jf_{i,j}fi,j​ 表示水桶 iii 在 2j2^j2j 次操作后，会在谁的手上
  • sumi,jsum_{i,j}sumi,j​ 表示水桶 iii 在 2j2^j2j 次操作后，有多少水
    转移方程为
    f[i, j] = f[f[i][j - 1], j - 1]
