# C

# 题目大意

如果一个数 nnn，能够找到唯一一对 x,y(0<x<y)x, y~(0 < x < y)x,y (0<x<y)，满足 x2+y2=nx^2+y^2=nx2+y2=n 的话，那么就称 nnn 为好数

现在，给你 nnn，请问，1→n1 \to n1→n 有多少好数？请你输出他们

# 数据范围

  • 1≤n≤1071 \le n \le 10^71≤n≤107

# 题解

这题实际上考的是预处理，我们只需要定义一个 cnt ，然后预处理出所有数可能出现的次数即可了，复杂度是 O(N2)O( {\sqrt{N}}^2)O(N​2)，可以接受

# D

# 题目大意

给定 nnn 个数，a1,…,ana_1, \ldots, a_na1​,…,an​，请问，满足以下条件的三元组 (i,j,k)(i, j, k)(i,j,k) 有几个？

  • 1≤i,j,k≤n1 \le i, j, k \le n1≤i,j,k≤n
  • ai:aj:ak=7:5:3a_i : a_j : a_k = 7 : 5 : 3ai​:aj​:ak​=7:5:3
  • min⁡(i,j,k)=jormax⁡(i,j,k)=j\min(i, j, k) = j \quad or \quad \max(i, j, k) = jmin(i,j,k)=jormax(i,j,k)=j

# 数据范围

  • 1≤n≤3×1051 \le n \le 3 \times 10^51≤n≤3×105
  • 1≤ai≤1091 \le a_i \le 10^91≤ai​≤109

# 题解

显然，根据 min⁡(i,j,k)=jormax⁡(i,j,k)=j\min(i, j, k) = j \quad or \quad \max(i, j, k) = jmin(i,j,k)=jormax(i,j,k)=j 这个条件，我们可以得知，当确定了 jjj 的位置时，i,ki, ki,k 要么都在 jjj 的左边，要么都在 jjj 的右边，由此可以衍生出做法

我们用 map 记录左右两边的数出现的个数，然后扫描 j:1→nj: 1 \to nj:1→n，如果 aja_jaj​ 是 555 的倍数，那么代表其可以当做中间的那个，我们只需要计算

cntl3×cntl7+cntr3×cntr7cnt_{l3} \times cnt_{l7} + cnt_{r3} \times cnt_{r7}cntl3​×cntl7​+cntr3​×cntr7​

累加即可，其中， l3 = r3 = 3 * (a[j] / 5), l7 = r7 = 7 * (a[j] / 5)

# E

# 题目大意

nnn 个人想放风筝，第 iii 个人站在 (ai,0)(a_i, 0)(ai​,0)，想把风筝放到 (bi,1)(b_i, 1)(bi​,1)
问这 nnn 个人，如果要求线不能交叉，能满足最多人放风筝的数量是多少？

# 数据范围

  • 1≤n≤2×1051 \le n \le 2 \times 10^51≤n≤2×105
  • 0≤ai,bi≤1090 \le a_i, b_i \le 10^90≤ai​,bi​≤109

# 题解

# 一些知识

为了写这道题，我们先介绍下最长上升子序列（LISLISLIS）和最长连续上升子序列（LCISLCISLCIS）

最长上升子序列：首先是 O(n2)O(n^2)O(n2) 的做法，定义 dp[i] 为以 iii 结尾的最长上升子序列的长度，很容易可以得到状态转移方程

dpi=max⁡j∈t,a[∀t]<a[i]dpj+1dp_i = \max_{j \in t, a[\forall t] < a[i]} dp_j + 1dpi​=j∈t,a[∀t]<a[i]max​dpj​+1

这样的是 O(n2)O(n^2)O(n2) 的，那有没有 O(nlog⁡n)O(n \log n)O(nlogn) 的做法呢？有的兄弟有的
我们可以额外维护一个 ddd 数组，did_idi​ 表示，长度为 iii 的最长上升子序列的末尾元素的最小值
显然，did_idi​ 是单调递增的，于是我们便可以用二分去找到位置

最长连续上升子序列：这个很简单了，就是扫一遍 O(n)O(n)O(n) 解决

# 最终解答

其实这题就是，先按 aaa 升序排，然后按 bbb 降序排，排完后做一个 LISLISLIS 就结束了
为什么按 BBB 降序排？
这是因为，如果按 BBB 升序排，那同一个起点，比如 (1,2),(1,3),(1,4)(1, 2), (1, 3), (1, 4)(1,2),(1,3),(1,4) 就会被算入 333，但实际上只算 111

# 题目大意
